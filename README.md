# matrix

- `С`, `C++`, `Python`

## Description
Релизации библиоткеки для работы с матрицами на языках C, C++, Python.
Реализованы функции работы с матрицами:
- Сложение
- Вычитание
- Умножение
- Умножение на число
- Определитель
- Дополнительная матрица
- Обратная матрица

#### Для C и C++
-собрать библиотеку
```
make matrix.a
```
-запустить тесты
```
make test
```
-получить отсчёт о тестах в виде html-страницы
```
make gcov_report
